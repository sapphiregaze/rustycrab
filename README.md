# Calculator

## Build

Run the build script with the below command:

```sh
./build.sh
```

The resulting binary can be found at `build/rc`.

## Usage

Run the below as an example to see the output (expressions are delimited by semicolons):

```sh
./build/rc "<file_path>"
```
